Amazon Review Sentiment Analysis

Model Overview

Our sentiment analysis model is built using a pre-trained distilBeRT model and fine-tuned on a large collection of labeled Amazon reviews. The model analyzes review text to determine whether the sentiment is positive or negative. With this app, you can feed it a product url from Amazon and it will tell you what percent of the reviews were positive.
